# Food Price Prediction

## Overview
This project predicts food prices using Machine Learning on real-world data obtained from the World Food Programme (WFP). The project includes data preprocessing, feature engineering, model training, and performance evaluation.

## Features
- Data cleaning and preprocessing
- Missing value imputation
- Outlier removal
- Feature engineering
- Time-based train-test split
- Food price prediction

## Machine Learning Models
- Linear Regression
- Decision Tree
- Random Forest

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
